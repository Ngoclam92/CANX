## 🔻 Emission Model

CANXO uses a predictable halving-based emission structure:

- Mining rewards distributed every block  
- Block reward decreases by 50% every 365 days  
- Inflation reduces over time  
- Supply expansion remains transparent and predictable  

This model encourages early participation while supporting long-term scarcity.
